01 / The Problem

The Reactive Trap

Standard Horizontal Pod Autoscalers (HPA) rely on CPU thresholds. By the time CPU hits 80%, traffic has already spiked. Compounded by pod cold-start delays, the cluster drops requests while waiting for new instances to boot.

To compensate, engineers over-provision resources, wasting massive amounts of money.

Deep Reinforcement Learning

Our RecurrentPPO agent observes a 23-dimensional state vector — including traffic acceleration, pending pods, and queue depth. It outputs replica deltas (-3 to +3) to continuously optimize a reward function balancing latency and node cost.
